# Assignment 3 Report

## Objective

To detect Amazon S3 buckets without server-side encryption using AWS Lambda and Boto3.

## AWS Services Used

- Amazon S3
- AWS Lambda
- IAM
- CloudWatch
- Boto3

## Steps Performed

1. Created encrypted and unencrypted S3 buckets.
2. Created IAM role.
3. Created Lambda function.
4. Implemented Boto3 script.
5. Tested Lambda manually.
6. Verified output through CloudWatch Logs.

## Output

The Lambda function identified buckets without server-side encryption.

## Conclusion

Successfully implemented automated monitoring of S3 bucket encryption using AWS Lambda and Boto3.
